# ArUco Indoor Localization System for Mobile Robotics (ROS 2 Humble)

Система локализации мобильного робота внутри помещений на основе потолочных ArUco-маркеров и монокулярной веб-камеры (Logitech C270). Проект разработан для среды **ROS 2 Humble** на базе **Ubuntu 22.04 LTS** (поддерживается как физическая система, так и виртуальное окружение WSL2).

Алгоритм вычисляет точное 3D-положение робота ($X, Y, Z$) и его пространственную ориентацию (в виде кватернионов) относительно глобальной карты маркеров и публикует данные в стандартный топик ROS 2.

## 🛠 Требования и зависимости

### 1. Операционная система и окружение
* **OS:** Ubuntu 22.04 LTS (Jammy Jellyfish)
* **ROS 2:** Humble Hawksbill ([Инструкция по установке ROS 2](https://docs.ros.org/en/humble/Installation.html))
