# ArUco Indoor Localization System for Mobile Robotics (ROS 2 Humble)

Система локализации мобильного робота внутри помещений на основе потолочных ArUco-маркеров и монокулярной веб-камеры (Logitech C270). Проект разработан для среды **ROS 2 Humble** на базе **Ubuntu 22.04 LTS** (поддерживается как физическая система, так и виртуальное окружение WSL2 под Windows 11).

Алгоритм вычисляет точное 3D-положение робота ($X, Y, Z$) и его пространственную ориентацию (в виде кватернионов) относительно глобальной карты маркеров и публикует данные в стандартный топик ROS 2.

## 🛠 Требования и зависимости

### 1. Операционная система и окружение
* **OS:** Ubuntu 22.04 LTS (Jammy Jellyfish)
* **ROS 2:** Humble Hawksbill ([Официальное руководство по установке](https://docs.ros.org/en/humble/Installation.html))

### 2. Зависимости Python
Для работы модулей компьютерного зрения необходимы расширенные пакеты OpenCV и утилиты работы с матрицами. Установите их в терминале Ubuntu:

```bash
sudo apt update
sudo apt install python3-pip python3-opencv
pip3 install numpy pyyaml opencv-contrib-python

⚠️ Важно: Пакет opencv-contrib-python обязателен, так как модули aruco для поиска маркеров находятся именно в репозитории расширений (contrib).
